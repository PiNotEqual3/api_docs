Updating a Tracked Link of an Email Campaign
============================================

Endpoint
--------

``POST /api/v2/email/<email_id>/trackedlinks/<link_id>``

Parameters
----------

.. list-table:: **Required Parameters**
   :header-rows: 1
   :widths: 20 20 40 40

       * - Name
         - Type
         - Description
         - Comments
       * - email_id
         - int
         - ID of the email, part of the URI
         -
       * - link_id
         - int
         - ID of the URL, part of the URI
         -
